## Documentation Principles

- Prefer explicit uncertainty over unsupported claims.
- Mark missing scientific metadata with `TODO`.
- Separate prototype behavior from validated model claims.
- Keep health and medical risk visible.
- Preserve historical context without using archived files as active guidance.
