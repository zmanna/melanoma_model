# Artifacts

This directory contains non-source artifacts that are useful for inspecting or reproducing the prototype.

## Layout

| Path | Purpose |
|---|---|
| `models/current/` | Packaged model artifact used by the local Flask app. |
| `sample_inputs/legacy_uploads/` | Historical uploads and test images preserved from the capstone prototype. |

Files in `sample_inputs/legacy_uploads/` should be reviewed before public release for privacy, licensing, and necessity.
